Overview

A machine learning approach to determine chemical shifts in NMR spectroscopy data

Fall 2019

Abstract

Nuclear magnetic resonance (NMR) spectroscopy is a method used to determine the chemical structure of a molecule based on the principle of resonance frequency: any nuclear spin in a given structure depends on its chemical local environment (i.e. shielding). Accurate methods using plane wave density functional theory (DFT) have been developed to calculate chemical shifts. However, these techniques are computationally costly and hence fail to calculate the latter for large or complex molecular structures. We hence thought that machine learning tools may potentially offer a new way to achieve this goal.

The aim of this project is then to build a powerful model to predict chemical shieldings of hydrogen atoms from a few hundreds of crystal structures from the Cambridge Structural Database (CSD) based on their local environment.

Methods

As we deal with a supervised learning problem (continuous target), linear regression was believed to be an appropriate class of methods to predict labels. To prevent overfitting, only regularized models were implemented, namely Ridge, Lasso and Elastic net. Their respective regularization parameters were all optimized through grid search.

In parallel, we implemented several multi-layer perceptron (MLP), each outputting a prediction which is compared to the actual true value. We build shallow and deep networks to emphasize the effect of depth on the prediction. We use the mean-square error (MSE) as loss and trained the network using the classical stochastic gradient descent (SGD) algorithm. We also incorporated dropout and batch normalization to reduce overfitting during the training.

Results

The final best cross-validated model, a neural network of 3 hidden layers, allowed to predict the chemical shielding from a complex structure with a MSE of 0.53 and a R2 coefficient of 0.96.

Tools: Python (Scikit-Learn) and PyTorch

Team: Yann Mentha, Maxime Epars, Gianni Giusto
