Projects

You will find below a nonexhaustive list of the projects I worked on during my Master at EPFL, in collaboration with many great teammates.

A machine learning approach to determine chemical shifts in NMR spectroscopy data

Fall 2019

As part of the Machine Learning course at EPFL (CS-433) and in collaboration with the Laboratory of Computational Science and Modeling (COSMO), Yann Mentha, Maxime Epars and I aimed to apply a machine learning pipeline to solid-state nuclear magnetic resonance (NMR) spectroscopy data.

Tools: Python (Scikit-Learn) and PyTorch

Repository

Higgs boson machine learning challenge

Fall 2019

Competition launched by the CERN to develop machine learning analysis techniques to detect the Higgs boson from a specific dataset. Yann Mentha, Maxime Epars and I took part in this challenge with the instruction to use only Python Numpy library.

Tools: Python (Numpy)

Repository

Graph methods for phenotype prediction in mice

Fall 2019

As an alternative to common genetic analysis methods such as GWAS or PheWAS, we used a network properties as well as graph singal processing methods to select and infer a phenotype based on the underlying genotype and/or protein expression level.

Tools: Python (Networkx, PyGSP)

Repository

Comparison of different CNN architectures on the classification of MNIST handwritten digits

Spring 2019

The goal of this project was to classify digits (from the MNIST dataset) with different network architectures. To do so, Hugo Michel, Daniel Tadros and I performed parameters optimization and took advantage of weight sharing.

Tools: PyTorch

Repository

Statistical modeling: linear mixed effect models

Spring 2019

As part of the Applied Biostatistics course, I carried out statistical analyses on a given dataset and finally proposed a model based on mixed models that best describes the underlying relations between the explanatory variables and the continuous response variable.

Tools: R

Detection of error-related potentials in EEG recorded signals

Spring 2019

From a brain-computer interaction (BCI) experiment recording brain signals through electroencephalography (EEG), we used signal processing to improve the signal-to-noise ratio of the recorded signal prior feeding it to a classifier aiming at assessing the presence of so-called error-related potentials (ErrPs).

Tools: Matlab, PyTorch
